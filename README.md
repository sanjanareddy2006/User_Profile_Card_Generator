# User_Profile_Card_Generator
A simple full-stack web application that allows users to create and display a profile card by entering their name, bio, and image URL. The project demonstrates how to handle POST requests and pass data between a frontend UI and a Node.js backend.

🚀 Features
User input form (name, bio, image URL)
Backend API using Express
POST request handling
Dynamic profile card rendering
Simple and clean UI
🛠️ Tech Stack
HTML
CSS
JavaScript (Frontend)
Node.js
Express.js
CORS
📁 Project Structure
profile-card-generator/ │ ├── server.js ├── package.json │ ├── public/ │ ├── index.html │ ├── style.css │ └── script.js
